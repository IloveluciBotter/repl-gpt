# Codex Instructions (Repo Guide)

## Goal
Work on this repo safely and in small, reviewable changes.

## How to run locally
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Start: `npm run start`

## Repo structure
- `Repl-GPT/` is the main app.
- Root folder contains workspace-level files.

## Rules
- Do not commit secrets. Never add `.env` files.
- Keep changes minimal and explain what changed and why.
- Prefer TypeScript-safe code and avoid breaking APIs.
