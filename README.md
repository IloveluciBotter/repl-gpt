# repl-gpt

This repository contains the HiveMind / Repl-GPT application.

## Quickstart

All commands should be run from the `Repl-GPT/` folder.

### Install
```bash
cd Repl-GPT
npm ci
```

### Dev
```bash
cd Repl-GPT
npm run dev
```

### Build
```bash
cd Repl-GPT
npm run build
```

### Start (production)
```bash
cd Repl-GPT
npm run start
```

## Notes
- Do not commit secrets (.env files). Use `.env.example` as a reference.
- CI builds the app from `Repl-GPT/`.
